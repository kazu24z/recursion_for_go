## 2点間の距離 easy
文字列として表された二点の正の座標の配列 pointsList が与えられるので、それぞれ整数型の距離に変換して返す calcDistanceList という関数を作成してください。

関数の入出力例
```
入力のデータ型： string[] pointsList

出力のデータ型： string[]

calcDistanceList(["100-200","2-313","422-21"]) --> [100,311,401]

calcDistanceList(["507-293","347-551","730-450","663-949","614-828","368-506"]) --> [214,204,280,286,214,138]

calcDistanceList(["316-89","128-969","732-784","861-256","863-495","25-375","215-833","350-443","925-115","356-540","805-404","491-615","584-797","573-557","571-334","370-200","157-474","747-828"]) --> [227,841,52,605,368,350,618,93,810,184,401,124,213,16,237,170,317,81]

```
